# B-ASS (Banek-as-a-Service)

Очень простое апи для получения анекдотов с baneks.site и baneks.ru.
Так же позволяет скачивать ссылки на картинки-мемы с одного из источников.
И рисовать текст на картинках используя встроенные методы работы с изображениями.

## Фичи

- [x] Случайный анекдот из двух источников (api/baneks/random).
- [x] Round Robin Балансировщик для /random, чтобы не наглеть.
- [x] Получение конкретного анекдота по slug из banek.site (api/baneks/:slug)
- [x] Получение случайных мемов (api/memes/random?year={year})
- [x] Создание мема при помощи POST запроса на /api/meme-generator

## Технологии

- [Golang 1.26](https://go.dev/)
- [Echo v5](https://echo.labstack.com/)

## Использование

- Установите пакеты с помощью команды:

```sh
go mod tidy
```

- Запустите:

```sh
go run cmd/main.go
```

- Или соберите:

```sh
go build cmd/main.go
```

**Так же можно через MAKE.**

`make run` для запуска
`make run-{os}` для билда

Cмотри *Makefile* для прочих команд.
